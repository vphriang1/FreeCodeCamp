Timestamp Microservice

Reads the queried date from the URL and parses it in Unix millisecond and UTC format.

Directions

      run "node server.js"
      go to localhost in address bar at process.env.PORT address
      add /api/timestamp/ to the address
      run it then add a date in the format of many digits representing milliseconds or something like 2015-12-01
